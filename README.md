# Movie Library — личная кинотека

**Автор:** Сайфутьяров Денис


**Описание:** Простое GUI-приложение для хранения и фильтрации информации о фильмах. Данные сохраняются в JSON-файл.
**Технологии:** Python 3.x, Tkinter.
**Управление версиями:** Git.

## Возможности:
- Добавление фильмов (название, жанр, год выпуска, рейтинг).
- Фильтрация по жанру и году.
- Валидация ввода (год — число; рейтинг от 0 до 10).
- Сохранение и загрузка данных из JSON.
- История изменений через Git.

## Примеры использования:
1. Запустите приложение: `python main.py`.
2. Введите данные о фильме и нажмите «Добавить фильм».
3. Используйте фильтры для поиска по жанру или году.
4. Данные сохраняются автоматически при добавлении фильма.
